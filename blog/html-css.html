Class

What are Classes, you ask? A Class is a way to tell your selector apart on a webpage, so it would be easier to style them in groups. For example, let's say you wanted several paragraphs on your web page to alternate between dark and light colours. To do this, you will give each paragraph tag either the Class dark or the Class light. Then you can style your two different Classes with whichever colour you like. You can use a Class multiple times on a page. You can use the same Class for many selectors. When using Classes, it is a good practice to name your classes appropriately. In the example for Classes, I used the name dark to emphasise that my paragraph will have a colour style.

Id

What are Ids, you ask? An Id is a way to label your elements on a webpage. Typically, Ids link to a different section on your webpage without navigating to a separate HTML page. For example, let's say you wanted several other contents on your web page, which have many scrolling miles. To do this, you will give each section an Id that identifies the area and then put that in a link tag. You can use an Id name only once on a page. Therefore, when using Ids, it is a good practice to name your Ids appropriately.
